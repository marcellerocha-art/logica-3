# Exercícios de Lógica com Python - Nível 3

Repositório com exercícios avançados de lógica de programação em Python. Este projeto aborda conceitos como validação de dados, manipulação de strings, loops complexos e processamento de informações em tempo real.

## 📚 Exercícios

| Arquivo | Descrição |
|---------|-----------|
| **monitor_temperatura.py** | Sistema de monitoramento de temperatura com histórico e alertas |
| **validar_senha.py** | Validador de senha com critérios de segurança |

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/marcellerocha-art/logica-3.git
cd logica-3
```

2. Execute um exercício:
```bash
python3 monitor_temperatura.py
```

ou

```bash
python3 validar_senha.py
```

3. Siga as instruções do programa e digite os valores conforme solicitado.

## 💻 Requisitos

- Python 3.6 ou superior
- Terminal/Prompt de comando

## 📝 Conceitos Abordados

- ✅ Validação de dados de entrada
- ✅ Manipulação avançada de strings
- ✅ Loops e condicionais complexos
- ✅ Listas e coleção de dados
- ✅ Processamento em tempo real
- ✅ Critérios de segurança
- ✅ Tratamento de exceções
- ✅ Funções com múltiplos parâmetros

## 🎯 Objetivo

Esses exercícios foram desenvolvidos para praticar conceitos mais avançados de lógica de programação, preparando para o desenvolvimento de aplicações mais complexas e orientadas a objetos.
